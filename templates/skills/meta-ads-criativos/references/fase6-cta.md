# FASE 6 — CTA

## Objetivo
Construir CTAs para os 5 bodys selecionados. 1 CTA por body.

## Regras do CTA
- **UMA ação.** Nunca duas.
- **Específico.** "Clique no link abaixo" > "Saiba mais".
- **Conectado ao hook.** Fecha o loop que o hook abriu.
- **Sem direção de performance.**

## Regra de Funil
- **Indireto:** CTA → "aula gratuita". Não menciona produto, preço, bônus, garantia.
- **Direto:** CTA → oferta. Pode mencionar produto, preço, bônus, garantia.

## Banco de Estratégias de CTA

Cada CTA deve usar **pelo menos 2 estratégias combinadas**. Variar entre os 5 scripts.

| Estratégia | Definição | Exemplo |
|---|---|---|
| Escassez / Urgência | Limita tempo ou disponibilidade | "por tempo limitado", "enquanto tiver no ar" |
| Exclusividade | Posiciona como restrito/privilegiado | "informação que só quem paga consultoria tem acesso" |
| Quebra de 4ª parede | Fala com produção ou referencia formato | "vou pedir pra produção colocar o link aqui embaixo" |
| Loop closing | Retoma linguagem exata do hook no CTA | Se hook fala de "X", CTA retoma "X" |
| Prova no CTA | Reforça resultado/número na chamada | "já ajudou X mil pessoas" |
| Privacidade / Sigilo | Para nichos sensíveis | "ninguém precisa saber", "acesso 100% sigiloso" |
| Antecipação de objeção | Neutraliza "será que é pra mim?" | "funciona pra qualquer idade" |
| Qualificação | Faz avatar sentir que merece/foi selecionado | "se ficou até aqui, essa aula foi feita pra você" |

## Formato de Entrega

```
SCRIPT [nº]
COMBO: [Ângulo A + Ângulo B]
FORMATO: [formato]

DIREÇÃO DE VÍDEO: [ação/cena/props dos primeiros segundos — da Fase 4.5]

HOOK: [hook adaptado]

BODY:
[script completo]

CTA: [chamada para ação]
ESTRATÉGIAS: [estratégias usadas]
```
