# PLAYBOOK — CRIATIVOS META ADS (MAPA DO PROCESSO)

> A lógica completa, bancos de ângulos/formatos e regras detalhadas estão na skill `meta-ads-criativos`. Este documento é só o mapa visual.

---

## FLUXO

```
F1 Inputs → F1.5 Estilo → F2 Ângulos → F3 Hooks → F4 Formatos → F4.5 Direção → F5 Body → F6 CTA → F7 Export
```

**Regra:** Uma fase por vez. Apresentar output. Aguardar aprovação. Avançar.

---

## O QUE CADA FASE FAZ

| Fase | Entrega | Depende de |
|---|---|---|
| **1 — Inputs** | 11 inputs coletados + Glossário do Avatar + Resumo validado | — |
| **1.5 — Estilo** | Padrões de tom dos ads validados + Calibrador (3 frases-exemplo) | F1 |
| **2 — Ângulos** | Filtragem do banco → 5 combos de 2 ângulos | F1 + F1.5 |
| **3 — Hooks** | 60 gerados → 20 filtrados → edição colaborativa → 20 fechados | F2 + Glossário + Calibrador |
| **4 — Formatos** | 20 hooks adaptados ao formato + distribuição aprovada | F3 + Formatos disponíveis (F1) |
| **4.5 — Direção** | Direção de vídeo (ação + props) para cada hook | F4 + Elementos de cena (F1) |
| **5 — Body** | 15 bodys gerados → 5 filtrados | F4/4.5 + Calibrador + Glossário + Tipo de funil |
| **6 — CTA** | 1 CTA por body (5 total) com estratégias combinadas | F5 + Tipo de funil |
| **7 — Export** | Documento .docx pronto pra produção | F6 |

---

## ATALHOS

- **"começar"** / **"fase 1"** → Inicia coleta de inputs
- **"próxima fase"** → Avança na sequência
- **"fase X"** → Vai direto (verifica dependências)
- **"exportar"** → Vai pra Fase 7

---

## REGRAS QUE ATRAVESSAM TUDO

1. **Tom** = dos ads validados (não da VSL)
2. **Recorte de dor** = UMA dor, filtra tudo
3. **Glossário** = checagem obrigatória em hooks/body/CTA
4. **Scripts** = texto falado, sem direção de performance
5. **Funil indireto** = produto/preço/bônus NUNCA aparecem
