# FASE 5 — BODY

## Objetivo
Construir o corpo do ad em volta do hook já formatado.

## Regra de Dosagem do Mecanismo

### Funil indireto (ad → aula gratuita):
- TEASING do mecanismo, nunca detalhar.
- Linguagem vaga: "a causa real", "o que tá travando por dentro", "X fatores".
- Conteúdo técnico fica pra aula. Ad só gera curiosidade.
- NUNCA fechar o loop no ad.

### Funil direto (ad → oferta):
- Pode revelar parcialmente o mecanismo.
- Ainda não fecha o loop (página de vendas fecha).
- Pode usar termos técnicos se o avatar os reconhecer.

### Em TODOS os casos:
Priorizar espelhamento da experiência do avatar ANTES de qualquer explicação. Proporção dor vs. mecanismo segue o padrão dos ads validados (Fase 1.5).

## Regra de Escrita

Scripts são **texto pra ser falado**. NÃO incluir:
- Direções de pausa
- Indicações de tom de voz
- Expressões faciais
- Ritmo de fala

## Etapa 5.1 — Geração

Escrever **15 scripts de body** (3 por combinação de ângulos). Para cada combo, escolher os hooks+formatos mais fortes daquela combinação como base dos 3 bodys (podem ser hooks diferentes ou variações sobre o mesmo hook).

**Tom:** seguir rigorosamente Fase 1.5. Consultar calibrador.

### Exemplo de estrutura:
```
HOOK: [hook formatado]
AMPLIFICAÇÃO: Expande dor/curiosidade (2-3 frases)
MECANISMO: Apresenta o "como" sem entregar tudo
PROVA: Elemento de credibilidade
TRANSIÇÃO PRO CTA: Frase-ponte
```
(Variações e estruturas dos ads validados são válidas.)

## Etapa 5.2 — Filtragem

Filtrar os **5 melhores bodys**.

### Critérios (em ordem de prioridade):
1. **Coerência hook → body** — Entrega o que o hook prometeu? Avança o loop sem fechar?
2. **Ritmo** — Lê fluido? Variação de frase curta/longa?
3. **Força da prova** — Concreto e crível? Ou genérico?
4. **Clareza do mecanismo** — Entende que EXISTE algo diferente?
5. **Naturalidade** — Soa como alguém falando no tom dos ads validados?

**Aguardar aprovação antes de avançar para Fase 6.**
