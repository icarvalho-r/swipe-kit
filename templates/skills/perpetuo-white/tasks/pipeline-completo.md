## Objetivo
Guiar o usuário através de todas as fases de construção de um perpétuo white, desde a definição da escada de produtos até a escala com 5 sistemas de ascensão rodando.

## Elicitação Inicial

Antes de iniciar, coletar do usuário:

```
1. Qual é o seu nicho? (investimentos, emagrecimento, marketing, etc.)
2. Você já tem um expert? (sim/não — se sim, quantos seguidores?)
3. Já tem produto principal? (curso, mentoria, etc. — qual ticket?)
4. Já faz lançamento? (sim/não — se sim, quantos por ano? qual faturamento?)
5. Já tem algum perpétuo rodando? (sim/não — se sim, qual ticket? qual ROI?)
6. Qual é o caixa disponível para testes? (R$5K, R$10K, R$20K+)
7. Quem vai escrever a VSL? (você, copywriter, coprodutor)
8. Quem vai operar o tráfego? (você, gestor, agência)
```

## Fases de Execução

### FASE 1: Escada de Produtos (Semana 1-2)
**Agent:** offer-architect

1. Definir front-end (R$150-300) a partir do produto principal
2. Criar checklist de 7 dias para resultado rápido
3. Definir upsell PRONTO (R$400-600)
4. Renomear back-end para "programa de acompanhamento" com tríade
5. Definir high-end (se aplicável)
6. Configurar cashback entre níveis

**Checkpoint:** Todos os produtos definidos com tickets, promessas e diferenciação entre níveis.

### FASE 2: VSL (Semana 2-4)
**Agent:** vsl-architect

1. Pesquisa de mercado (3 perguntas + YouTube)
2. Escrever oferta (primeiro!)
3. Escrever mecanismo do problema (história do expert + dados)
4. Escrever mecanismo da solução (O QUE + POR QUÊ + 2 caminhos)
5. Escrever 5 leads (negativa, positiva, história, depoimento, autoridade)
6. Gravar VSL (teleprompter + leads em formatos variados)
7. Configurar na VTurb (botão desconto, thumbnail, teste AB)

**Checkpoint:** VSL gravada, 5 leads configuradas para teste AB, botão desconto programado.

### FASE 3: Funil Imediato (Semana 3-4)
**Agent:** offer-architect + ascension-builder

1. Configurar checkout (order bump se aplicável)
2. VSL de upsell PRONTO pós-checkout
3. Downsell se não comprar
4. Formulário de matrícula no primeiro módulo
5. Aula de boas-vindas com depoimentos + escada
6. Automação Make → ManyChat para convite webinário diário

**Checkpoint:** Funil completo testado end-to-end. Pessoa compra → upsell → formulário → convite webinário.

### FASE 4: Criativos + Tráfego (Semana 4-6)
**Agent:** creative-engineer + traffic-scaler

1. Produzir mínimo 10 criativos "sóbrios" (3+ formatos)
2. Extrair 10 ângulos da VSL para criativos
3. Se tem orgânico viral: cortar + CTA
4. Configurar ABO Gaveta (meio ticket por criativo)
5. Rodar 2-3 dias, validar os que vendem com ROI
6. Subir validados para CBO pré-escala

**Checkpoint:** Pelo menos 2-3 criativos validados. ROI front ≥ 1.3. Volume de vendas diário estável.

### FASE 5: Ascensões (Semana 6-12)
**Agent:** ascension-builder

1. Webinário diário gravado + automação ManyChat (mesmo dia)
2. Workshop semanal ao vivo (80% conteúdo + 20% pitch curto)
3. Time comercial abordando leads do formulário
4. Próximo lançamento planejado com base de front-end

**Checkpoint:** ROI subindo de 1.7 (front) para 2.5+ (com ascensões). Meta: 3.0+.

### FASE 6: Otimização Contínua (Ongoing)
**Agents:** todos

1. Testar 10 criativos/dia, 5 leads a cada 15 dias
2. Escalar 20-30%/dia campanhas validadas
3. Worldwide em português
4. Apertar parafusos: upsell → workshop → webinário → formulário
5. Meta: R$50-100K lucro/dia

## Veto Conditions
- NAO escalar sem mínimo 5 leads testadas na VSL
- NAO escalar sem upsell imediato PRONTO (não curso)
- NAO diagnosticar VSL como ruim com menos de 10 criativos testados
- NAO duplicar campanhas (escalar gradualmente 20-30%/dia)
- NAO fazer mecanismo mirabolante para nicho white

## Output Esperado
Funil perpétuo white completo rodando com:
- Front-end vendendo via VSL + tráfego direto
- Upsell PRONTO com 15%+ conversão
- 5 sistemas de ascensão ativos
- ROI final ≥ 2.5 (meta 3.0+)
- Escala crescendo 20-30%/dia
