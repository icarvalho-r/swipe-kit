## Criterios de Aprovacao

- **READY FOR WRITING:** Todos os checks BLOCKING passam
- **NEEDS WORK:** Qualquer check BLOCKING falha → completar fase correspondente
- **INCOMPLETE:** 5+ checks BLOCKING falham → voltar ao planejamento

## VETO Conditions (nao iniciar escrita)

1. Mecanismo nao definido → VETO para thesis-writer
2. Big Idea sem os 3 elementos → VETO para thesis-writer
3. Menos de 3 tendencias de mercado → VETO (reconsiderar nicho)
4. Emotional Story sem fundo do poco → VETO para story integration
5. Discovery Story desconectada da Big Idea → VETO para story integration
